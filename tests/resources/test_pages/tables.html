Tables (etalon)

This page contains four table variants for Hyperion table component tests:
1) standard HTML table (no header)
2) standard HTML table (with header)
3) custom div/grid table (editable: first text, middle inputs, last actions)
4) custom div/grid table (mixed: text + icon + keyed input column + actions)
