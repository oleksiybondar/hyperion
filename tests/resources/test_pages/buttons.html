Buttons (etalon)

This page contains two button variants for Hyperion component tests:
1) standard button (text on the clickable element)
2) decoupled label (click target and label are separate elements)

Standard button

Click target and label are the same <button>. Text is updated on each click.

Decoupled label

Click target is an (optionally empty) <button>, while visible text is in a separate label element. This simulates backends where the containerâ€™s .text can be empty.
